多行列表项响应式布局

使用 grid 布局，涉及到 `repeat` 函数的使用，请参考 Grids 之登堂入室的 repeat 部分

1、间距固定

容器（主体内容）最大宽度为1400px，间距固定，列表项有个最小宽度，尽可能多的生成列表项，剩余容器宽度均分给列表项。Demo 如下（可改变宽度查看效果）：

CSS 代码示意如下：

:root {
  --grid-item-width: 200px;
  --grid-gap: 15px;
}

/* 
* 使用最小列宽度循环生成尽可能多的列，剩余不够一列的宽度均分给各列
*/
.ml-grid-list {
  display: grid;
  grid-template-columns: repeat(
    auto-fit,
    minmax(var(--grid-item-width), 1fr)
  );
  gap: var(--grid-gap);
}

2、列表项宽度固定

容器（主体内容）最大宽度为1400px，列表项固定宽度，尽可能多的生成列表项，剩余容器宽度均分给间距。Demo 如下：

CSS 代码示意如下：

:root {
  --grid-item-width: 200px;
  --grid-gap: 15px;
}

/* 
* 使用固定列宽度循环生成尽可能多的列
* 设置水平分布为 space-between
*/
.ml-fixed-grid-list {
  display: grid;
  grid-template-columns: repeat(auto-fill, var(--grid-item-width));
  justify-content: space-between;
  gap: var(--grid-gap);
}
