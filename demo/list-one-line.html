单行列表项响应式布局

使用 flex 布局，要点： flex 剩余空间分配的使用；水平分布的使用

1、列表项等分

直接设置列表项 flex: 1;， 即可实现等分。Demo 如下：

1
2
3
4
5

CSS 代码示意如下：

.ml-flex-list {
  display: flex;
}
.ml-flex-item {
  flex: 1; /* 等分 */
}

2、列表项宽度固定

列表项固定宽度，剩余容器宽度均分给间距。Demo 如下：

1
2
3
4
5

CSS 代码示意如下：

.ml-fixed-flex-list {
  display: flex;
  justify-content: space-between;
}
.ml-fixed-flex-list .ml-flex-item {
  width: 100px;
}
